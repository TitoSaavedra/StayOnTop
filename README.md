# StayOnTop

Keep any window always on top — no matter what you click or open.

**[Download latest version](https://github.com/TitoSaavedra/StayOnTop/releases/latest)**

---

## What it does

StayOnTop lets you pin any running window so it stays visible above everything else on your screen. Useful for keeping a reference, a calculator, a chat, a video — anything — always in view while you work.

---

## Features

**Pin windows**
- See all running windows in a list and pin any of them with one click
- Pinned windows stay on top even when you switch apps
- Unpin individually or all at once

**Control each pinned window**
- **Opacity** — make it as transparent as you want (10% to 100%)
- **Click-through** — let your mouse clicks pass through the window so you can interact with what's behind it
- **Position & size** — set exact X, Y, width and height from the app

**Image overlay**
- Pin any image directly on your screen, with opacity and scale control

**Global hotkey**
- Set a keyboard shortcut to pin or unpin the window you're currently using, without touching the app

**Light & dark theme**
- Switch between dark and light mode in Settings → General

**Auto-update**
- The app checks for updates on launch and notifies you in Settings → About

**Other**
- Starts with Windows (optional)
- Keeps running in the system tray when closed
- Available in English, Español and Português (BR)

## Installation

1. Go to the **[Releases](https://github.com/TitoSaavedra/StayOnTop/releases/latest)** page
2. Download the `.msi` installer
3. Run it — Windows may show a SmartScreen warning the first time; click **More info → Run anyway**
4. StayOnTop appears in the system tray

---

## How to use

**Pin a window**
1. Open StayOnTop from the tray
2. The list shows all running windows
3. Click the pin icon next to any window — it immediately stays on top

**Use the hotkey**
1. Go to **Settings → Hotkeys**
2. Click the key field and press your desired combination (e.g. `Ctrl+Shift+T`)
3. From anywhere, press that shortcut to pin or unpin the window you're currently using

**Adjust a pinned window**
- Use the **opacity slider** to control transparency
- Enable **Click-through** if you want the window to be visible but not intercept your clicks
- Click **Position & size** to type in exact coordinates or dimensions

**Pin an image**
- Go to the **Images** tab and click **Add Image**
- Adjust opacity and scale as needed

---

## Updates

When a new version is available, a dot appears on the **Settings** tab. Go to **Settings → About** and click **Install update** — the app will download and restart automatically.
